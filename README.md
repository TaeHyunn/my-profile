# 조태현 개발자 이력서

Vue 중심 프론트엔드 엔지니어 조태현의 한국어 이력서 사이트입니다. 관리자웹, 입주민앱, 경비원웹에서 실제로 담당한 기능과 운영 개선 경험을 정리했습니다.

공개 사이트: <https://taehyunn.github.io/my-profile/>

## 기술 구성

- Vue 3와 Composition API
- Vite
- Vitest와 Vue Test Utils
- 순수 CSS 반응형 레이아웃
- GitHub Actions와 GitHub Pages

## 로컬 실행

Node.js 22 이상을 권장합니다.

```bash
npm install
npm run dev
```

## 검증

```bash
npm test -- --run
npm run lint
npm run build
```

## 배포

`main` 브랜치에 변경 사항을 푸시하면 GitHub Actions가 테스트, 린트, 빌드를 순서대로 실행한 뒤 GitHub Pages에 배포합니다. 저장소의 Pages 설정에서 소스를 `GitHub Actions`로 지정해야 합니다.

## 정보 공개 원칙

프로젝트 설명은 실제 커밋 이력에서 확인된 기여만 사용했습니다. 비공개 저장소 주소, 사내 코드, 내부 화면, 임의로 만든 성과 수치는 포함하지 않습니다.
